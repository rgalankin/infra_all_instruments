# IDX (IIDX.RU)

## ОСНОВНАЯ ИНФОРМАЦИЯ

- **Категория:** API - Identity Verification
- **Статус:** pending (заявка отправлена на info@iidx.ru)
- **Критичность:** средняя (KYC, верификация клиентов)
- **Назначение:** верификация клиентов, проверка личности и документов
- **Платформа:** cross-platform (предположительно REST API)
- **Сайт:** https://iidx.ru
- **Контакт:** info@iidx.ru

## ПОДКЛЮЧЕНИЕ

- **Auth:** ожидается информация от провайдера
- **Credentials:** нет (ожидается подключение)

## ВОЗМОЖНОСТИ (ПРЕДПОЛАГАЕМЫЕ)

| Функция | Описание | Применимость |
|---------|----------|-------------|
| Проверка паспорта | Верификация данных паспорта | Высокая |
| Проверка личности | Идентификация по фото/документу | Средняя |
| KYC | Комплексная проверка клиента | Высокая |

## СКРИПТЫ

Скрипт не создан. Ожидается подключение и документация API.

## СТОИМОСТЬ

Уточняется. Предположительно платный сервис.

## СЦЕНАРИИ ИСПОЛЬЗОВАНИЯ

### P1: Верификация клиентов при приёме
- Проверка подлинности паспорта
- Сверка фото с документом

### P2: Anti-fraud
- Обнаружение поддельных документов
- Проверка при оформлении кредитных заявок
