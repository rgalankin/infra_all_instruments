# API-PARSER.RU

## ОСНОВНАЯ ИНФОРМАЦИЯ

- **Категория:** API - Government Data Aggregator
- **Статус:** pending (заявка отправлена на info@api-parser.com)
- **Критичность:** высокая (доступ к данным госструктур для банкротства)
- **Назначение:** единый REST-интерфейс к данным ФССП, ГИБДД, ФНС, МВД, РСА, ЕАИСТО, ФНП
- **Платформа:** cross-platform (REST API)
- **Сайт:** https://api-parser.ru
- **Контакт:** info@api-parser.com

## СЦЕНАРИИ ИСПОЛЬЗОВАНИЯ

### P0: Проверка клиентов по ФССП
- Автоматическая проверка наличия исполнительных производств
- Мониторинг изменений по клиентам в процессе банкротства

### P1: Комплексная проверка контрагентов
- Одновременный запрос по нескольким реестрам
- Формирование отчёта для юриста
