# ФССП API

## ОСНОВНАЯ ИНФОРМАЦИЯ

- **Категория:** API - Government Services (ФССП России)
- **Статус:** unavailable (портал api-ip.fssp.gov.ru возвращает 404)
- **Критичность:** высокая (core-бизнес - банкротство физических лиц)
- **Назначение:** доступ к Банку данных исполнительных производств ФССП
- **Платформа:** cross-platform (REST API)
- **URL:** https://api-ip.fssp.gov.ru
- **Контакт:** admin@fssp.gov.ru (заявка отправлена)

## ПОДКЛЮЧЕНИЕ

- **Auth:** API key (выдаётся по заявке через портал ФССП)
- **Credentials:** нет (портал недоступен)
- **Регистрация:** https://api-ip.fssp.gov.ru/account/register

## ДОСТУПНЫЕ ДАННЫЕ (ИЗ ДОКУМЕНТАЦИИ)

| Данные | Описание | Применимость |
|--------|----------|-------------|
| Исполнительные производства | По ФИО + дата рождения | Критическая |
| Суммы задолженности | Общая сумма долга по ИП | Критическая |
| Статус ИП | Открыто/закрыто/на исполнении | Высокая |
| Данные пристава | ФИО, телефон, отдел | Средняя |

## ENDPOINTS (ИЗ ДОКУМЕНТАЦИИ)

```
GET /api/v1.0/search/physical - поиск физлиц
GET /api/v1.0/search/legal    - поиск юрлиц
GET /api/v1.0/search/ip       - поиск по номеру ИП
GET /api/v1.0/result           - получение результата (асинхронный)
```

**Модель работы:** асинхронная
1. Отправить поисковый запрос -> получить task_id
2. Опросить результат по task_id -> получить данные

## СКРИПТЫ

Скрипт не создан. Портал API недоступен.

## СТОИМОСТЬ

Бесплатно (государственный сервис). Лимиты:
- Обычный ключ: несколько запросов в минуту
- Расширенный ключ: по согласованию с ФССП

## ТЕКУЩАЯ ПРОБЛЕМА

Портал https://api-ip.fssp.gov.ru возвращает 404. Предположительно:
- Техническое обслуживание
- Миграция на новую версию API
- Заявка отправлена на admin@fssp.gov.ru

**Альтернативный доступ через:**
- SpectrumData (платный, агрегирует данные ФССП)
- api-parser.ru (платный, агрегирует данные ФССП)
- fssp.gov.ru (веб-интерфейс, без API)

## СЦЕНАРИИ ИСПОЛЬЗОВАНИЯ

### P0: Проверка клиентов на наличие исполнительных производств
- Обязательная проверка при приёме заявки на банкротство
- Определение общей суммы задолженности клиента
- Выявление количества и типов ИП

### P0: Мониторинг ИП в процессе банкротства
- Отслеживание изменений статуса ИП во время процедуры
- Контроль закрытия ИП после завершения банкротства

### P1: Автоматический скоринг клиентов
- Интеграция данных ФССП в CRM-карточку контакта
- Автоматический расчёт сложности дела

## СВЯЗИ

### Альтернативы
- **SpectrumData** - платный B2B API с доступом к данным ФССП
- **api-parser.ru** - агрегатор API госструктур
- **fssp.gov.ru** - веб-интерфейс для ручной проверки

### Интеграции
- **Bitrix24 CRM** - автозаполнение данных ИП в карточке клиента
- **Sales Agent (Максим)** - автопроверка при входящем обращении
- **n8n** - автоматические проверки в workflows
