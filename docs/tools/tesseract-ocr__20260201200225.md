# Tesseract OCR

## Основная информация

- **Категория:** AI → OCR / image processing
- **Версия:** 5.5.2
- **Статус:** активно
- **Критичность:** средняя
- **Назначение:** OCR (оптическое распознавание текста) для обработки изображений и PDF документов

## НАСТРОЙКИ

### ОСНОВНЫЕ ПАРАМЕТРЫ
- **Тип:** CLI инструмент (через Python библиотеку `pytesseract`)
- **Платформа:** macOS (локальная установка)
- **Режим работы:** production
- **Языки:** русский, английский (и другие через языковые пакеты)

### ВЕРСИЯ И ОКРУЖЕНИЕ
- **Python библиотека:** `pytesseract`
- **Доступ:** через командную строку `tesseract`
- **Платформа:** macOS

## ИСПОЛЬЗОВАНИЕ

### ПОДДЕРЖИВАЕМЫЕ ФОРМАТЫ
- **Входные:** JPG, JPEG, PNG, PDF (через pdf2image)
- **Выходные:** текст (plain text)

### ОСОБЕННОСТИ
- **Базовый OCR** - используется как первый этап обработки
- **Работает с PDF** - через конвертацию страниц в изображения
- **Универсальность** - работает с любыми форматами изображений

## ИНТЕГРАЦИИ

### PYTHON СКРИПТЫ
- **`process_images.py`** - обработка изображений через Tesseract
- **`process_pdf_scans.py`** - обработка PDF сканов через Tesseract
- **`process_all_ocr_improvements.py`** - сравнение с PaddleOCR и улучшение через LLM
- **`compare_and_merge_ocr.py`** - сравнение результатов Tesseract и PaddleOCR

### СВЯЗИ С ДРУГИМИ КОМПОНЕНТАМИ
- **PaddleOCR** - используется вместе для сравнения и улучшения результатов
- **Ollama (qwen3:30b)** - LLM для улучшения результатов OCR
- **pdf2image** - для конвертации PDF в изображения

## ПРОБЛЕМЫ И ЗАМЕЧАНИЯ

- **Точность для русского языка** - ниже чем у PaddleOCR
- **Требует языковые пакеты** - для поддержки русского языка нужен отдельный пакет
- **Скорость** - может быть медленнее чем PaddleOCR для больших документов

## ПОТЕНЦИАЛ УЛУЧШЕНИЯ

- Использование в паре с PaddleOCR для максимальной точности
- Улучшение через LLM для исправления ошибок OCR
