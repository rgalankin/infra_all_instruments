# LIVEDUNE

## ОСНОВНАЯ ИНФОРМАЦИЯ

- **Категория:** API - Social Media Analytics
- **Статус:** активно (тариф активирован, тестируется)
- **Критичность:** средняя (маркетинг, аналитика соцсетей)
- **Назначение:** аналитика соцсетей: метрики аккаунтов, охваты, вовлечённость, демография аудитории
- **Платформа:** Web (REST API)
- **Сайт:** https://pro.livedune.com
- **API Base URL:** https://api.livedune.com
- **Документация:** https://api.livedune.com/docs/
- **OpenAPI:** https://api.livedune.com/docs/openapi.json
- **Версия API:** 1.0.9

## ПОДКЛЮЧЕНИЕ

- **Auth:** Query parameter `access_token`
- **Credentials:** `.secrets/api-keys/livedune.json`
- **Env variable:** `LIVEDUNE_API_KEY`
- **Пагинация:** параметр `after` (timestamp), лимит 100 записей на страницу

## ENDPOINTS

| Endpoint | Описание |
|----------|----------|
| `GET /accounts` | Список аккаунтов в дашборде |
| `GET /projects` | Список проектов |
| `GET /accounts/{id}/analytics` | Сводка метрик за период |
| `GET /accounts/{id}/history` | Ежедневная история метрик |
| `GET /accounts/{id}/audience/{mode}` | Демография аудитории (all/age/gender) |
| `GET /accounts/{id}/posts` | Список постов с метриками |
| `GET /accounts/{id}/stories` | Список сторис с метриками |
| `GET /accounts/{id}/video` | Список видео с метриками |
| `GET /info` | Информация о тарифе и лимитах |

## МЕТРИКИ

- Подписчики (прирост, отписки)
- Охваты (reach, impressions)
- Engagement Rate (ER, ER по просмотрам)
- Лайки, комментарии, репосты (среднее и суммарное)
- Демография аудитории (пол, возраст)
- Данные по постам, сторис, видео

## СКРИПТ

**Путь:** `~/Documents/ai_agents/scripts/livedune-stats.py`

**Зависимости:** Python 3, requests

```bash
# Список аккаунтов
python3 scripts/livedune-stats.py --action accounts

# Аналитика за 30 дней
python3 scripts/livedune-stats.py --action analytics --account 777165 --period 30d

# Демография аудитории
python3 scripts/livedune-stats.py --action audience --account 777165 --mode gender

# Посты за неделю
python3 scripts/livedune-stats.py --action posts --account 777165 --period 7d --limit 10

# Информация о тарифе
python3 scripts/livedune-stats.py --action tariff
```

**Возможности скрипта:**
- 9 действий (accounts, projects, analytics, history, audience, posts, stories, videos, tariff)
- Пагинация для больших наборов данных
- Форматы вывода: json, table
- Парсинг периодов: 7d, 30d, 90d, YYYY-MM-DD:YYYY-MM-DD

## СЦЕНАРИИ ИСПОЛЬЗОВАНИЯ

### P1: Мониторинг SMM-эффективности
- Еженедельный отчёт по охватам, вовлечённости, приросту подписчиков
- Сравнение периодов для оценки эффективности контент-стратегии

### P2: Анализ аудитории
- Демографический профиль подписчиков (пол, возраст)
- Сегментация для таргетирования контента

### P2: Анализ контента
- Определение лучших постов по метрикам
- Выявление оптимального времени публикации
