# SPECTRUMDATA B2B API

## ОСНОВНАЯ ИНФОРМАЦИЯ

- **Категория:** API - Compliance & Verification (проверка контрагентов, физлиц)
- **Статус:** in_progress (аккаунт зарегистрирован, ожидается активация)
- **Критичность:** высокая (ключевой инструмент для due diligence и KYC)
- **Назначение:** проверка контрагентов и физлиц по базам госструктур (ФССП, ФНС, ЕФРСБ, МВД, суды)
- **Платформа:** cross-platform (REST API)
- **Сайт:** https://spectrumdata.ru
- **Документация:** https://b2b-api.spectrumdata.ru/docs/about/overview
- **Swagger:** https://b2b-api.spectrumdata.ru/swagger-ui/index.html

## ПОДКЛЮЧЕНИЕ

- **Auth:** Токен AR-REST (генерируется на клиенте)
- **Алгоритм токена:** MD5(password) -> base64 -> salt(stamp:age:pass_hash) -> MD5 -> base64
- **Header:** `Authorization: AR-REST <token>`
- **Credentials:** `.secrets/api-keys/spectrumdata.json`
- **Base URL:** `https://b2b-api.spectrumdata.ru`

## ДОСТУПНЫЕ ДАННЫЕ

| Источник | Описание | Применимость |
|----------|----------|-------------|
| ФССП | Исполнительные производства | Высокая (банкротство) |
| ФНС | ИНН, ЕГРЮЛ, ЕГРИП | Высокая (верификация) |
| ЕФРСБ | Реестр банкротств | Высокая (core-бизнес) |
| МВД | Розыск, паспорта | Средняя (KYC) |
| Суды | Судебные дела | Высокая (банкротство) |

## СКРИПТЫ

Скрипт пока не создан. Ожидается активация API-аккаунта.

## СТОИМОСТЬ

Платный сервис. Тарификация по количеству запросов. Точные условия уточняются.

## СЦЕНАРИИ ИСПОЛЬЗОВАНИЯ

### P0: Автоматическая проверка клиентов при приёме
- Проверка наличия исполнительных производств (ФССП)
- Проверка ИНН и статуса в ФНС
- Проверка в реестре банкротов (ЕФРСБ)

### P1: Due Diligence контрагентов
- Полная проверка юрлиц перед сотрудничеством
- Мониторинг изменений (суды, банкротство)

## РИСКИ

- **risk_tier:** medium
- **data_allowed:** проверка публичных реестров по ФИО/ИНН с согласия субъекта
- **data_forbidden:** массовый сбор данных без оснований
- **known_failure_modes:**
  1. Токен генерируется клиентским алгоритмом (нестандартная авторизация)
  2. API-аккаунт может требовать верификации юрлица

## СВЯЗИ

### Альтернативы
- **api-parser.ru** - агрегатор госданных (альтернатива)
- **ФССП API** - прямой доступ к данным ФССП (бесплатно, нестабильно)

### Интеграции
- **n8n** - HTTP Request node для автоматических проверок
- **Bitrix24 CRM** - обогащение карточки контакта данными проверки
- **Sales Agent** - автопроверка клиентов при входящем обращении
