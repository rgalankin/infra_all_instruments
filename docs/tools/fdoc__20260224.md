# F.DOC

## ОСНОВНАЯ ИНФОРМАЦИЯ

- **Категория:** EDM - Electronic Document Management
- **Статус:** pending (заявка на подключение оставлена)
- **Критичность:** средняя (юридически значимый документооборот)
- **Назначение:** электронный документооборот, обмен юридически значимыми документами с контрагентами
- **Платформа:** Web
- **Сайт:** https://fdoc.ru

## ПОДКЛЮЧЕНИЕ

- **Auth:** ожидается информация от провайдера
- **Credentials:** нет (ожидается подключение)

## ВОЗМОЖНОСТИ (ПРЕДПОЛАГАЕМЫЕ)

| Функция | Описание | Применимость |
|---------|----------|-------------|
| Обмен документами | Отправка/получение электронных документов | Высокая |
| ЭЦП | Подпись документов электронной подписью | Высокая |
| Роуминг | Обмен между операторами ЭДО | Средняя |
| Архив | Хранение документов | Средняя |

## СКРИПТЫ

Скрипт не создан. Ожидается подключение и документация API.

## СТОИМОСТЬ

Платный сервис. Тарификация зависит от объёма документооборота.

## СЦЕНАРИИ ИСПОЛЬЗОВАНИЯ

### P1: Электронный документооборот с контрагентами
- Обмен актами, счетами, договорами в электронном виде
- Юридическая значимость (ЭЦП)

### P2: Интеграция с бухгалтерией
- Автоматический приём входящих документов
- Архивное хранение

## СВЯЗИ

### Альтернативы
- **Сбис (СБИС)** - уже используется для отчётности
- **Контур.Диадок** - популярный оператор ЭДО
- **1С:ЭДО** - встроенный в 1С

### Интеграции
- **Bitrix24** - уведомления о входящих документах
- **DocOps Agent (Павел)** - учёт в реестре документов
