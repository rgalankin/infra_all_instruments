# Интеграция: VK API

## Описание

Интеграция с VK API для публикации постов, загрузки фото и управления сообществами. Используется User Token (бессрочный) для wall.post и Community Token для остальных операций.

## Архитектура

```
OpenClaw / n8n → HTTPS REST → api.vk.com/method/ → VK
```

## Конфигурация

- **Протокол:** HTTPS REST
- **Endpoint:** `https://api.vk.com/method/`
- **Аутентификация:** OAuth (User Token — бессрочный, Community Token)
- **Секреты:** `/opt/hub/secrets/vk_tokens.json` (User Token + Community Token)
- **Client ID (OAuth):** 6121396
- **VK User ID:** 1219324

### Основное сообщество

- **ID:** 209257910
- **Slug:** roman_pro_finansy
- **Подписчики:** ~70

### Доступные методы

- `wall.post` — публикация постов (только с User Token, НЕ Community Token)
- `photos.upload` — загрузка фото
- `groups.edit` — описание, аватар, статус сообщества

### Ограничения

- Виджеты-кнопки под обложкой сообщества — только через web UI, не через API
- Community Token НЕ может выполнять `wall.post`
