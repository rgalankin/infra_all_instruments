# Интеграция: Avito API

## Описание

Интеграция с Avito API для работы с мессенджером (чтение чатов, отправка сообщений), статистикой объявлений и профилем аккаунта. Аккаунт «Кредитный советник Роман Галанкин» с 16 объявлениями по кредитным услугам в 16 городах.

## Архитектура

```
OpenClaw / n8n → HTTPS REST → api.avito.ru → Avito
                                    │
                              webhook (new messages)
                                    │
                              n8n.fingroup.ru/webhook/avito
```

## Конфигурация

- **Протокол:** HTTPS REST
- **Endpoint:** `https://api.avito.ru/`
- **Аутентификация:** OAuth 2.0, grant_type=client_credentials, токен на 24 часа
- **Секреты:** `/opt/hub/secrets/avito_credentials.json` (client_id + client_secret)
- **Аккаунт:** Кредитный советник Роман Галанкин (credit@fingroup.ru)
- **User ID:** 388160200

### Доступные методы

- `GET /core/v1/accounts/self` — профиль аккаунта
- `GET /messenger/v2/accounts/{id}/chats` — чаты мессенджера
- Отправка сообщений в чаты
- `POST /stats/v1/accounts/{id}/items` — статистика объявлений
- Вебхуки (подписка на события)

### Ограничения

- Создание/редактирование объявлений — нужен Autoload API (отдельный тариф)
- Управление профилем — только через web UI
- Каждый профиль Авито требует отдельный API-ключ

### Текущие данные

- 16 объявлений «Помощь в получении кредита» (16 городов)
- 100+ чатов в мессенджере

### Вебхук

- **URL:** `https://n8n.fingroup.ru/webhook/avito`
- **События:** входящие сообщения
