## ДЕТАЛЬНЫЕ ОПИСАНИЯ

### Replicate (replicate.com) - РЕКОМЕНДУЕТСЯ

**Технология:** Flux.1 [dev], [schnell], [pro]. Обучение через ostris/ai-toolkit.

- Загрузка 5-20 фото, обучение ~2 минуты на H100
- Стоимость обучения: ~$1.85 (1000 шагов, 20 фото)
- Генерация: Flux schnell $0.003/фото, dev $0.03/фото, pro $0.055/фото
- REST API, Python/Node.js SDK, webhook-уведомления
- LoRA-веса можно скачать
- Холодный старт 10-30 сек

### fal.ai - РЕКОМЕНДУЕТСЯ

**Технология:** Flux 1/2, собственный Flux 2 Turbo.

- Portrait Trainer (специализирован для лиц): $2.40 за 1000 шагов
- Flux 2 Trainer: $8.00 за 1000 шагов
- Flux Kontext Trainer: $1.25-2.50
- Генерация: schnell $0.003/мп, dev $0.025/мп, Flux 2 Turbo $0.008/фото
- REST API, Python/JS SDK
- Заявлено 4x быстрее конкурентов по inference
- Доступ к Kling 2.5, Wan 2.5, Veo 3 для видео (без fine-tuning лица)

### Black Forest Labs (bfl.ai) - МАКСИМАЛЬНОЕ КАЧЕСТВО

**Технология:** Flux Pro (оригинал, закрытая модель).

- Создатели Flux - доступ к Pro-версии (качественнее dev/schnell)
- Обучение: $2-6 (по итерациям: до 150 = $2, 150-500 = $4, 500+ = $6)
- Генерация: $0.025-0.06/фото
- Fine-tuning на Pro даёт лучшие результаты
- Модель нельзя скачать (vendor lock-in)

### Astria.ai - ЛУЧШИЙ ДЛЯ ЛИЦ

**Технология:** Flux, SDXL, собственные оптимизации. Специализация на fine-tuning.

- Самое дешёвое обучение: $1.50 (1200 шагов)
- FaceID технология для консистентности лица
- Специальные пресеты: Face, Product, Style
- Генерация: $0.23 за 8 фото (~$0.03/фото)
- Multiperson (несколько лиц в одной модели)
- Базовая поддержка Image2Video

### Leonardo.ai

- Собственная модель Phoenix + Flux, SDXL
- Бесплатный тариф: 150 кредитов/день
- Fine-tuning через кредиты
- Хороший для экспериментов, слабый API для продакшена

### HeyGen (heygen.com) - ЛУЧШИЙ ДЛЯ ВИДЕО

**Технология:** Avatar IV, собственная.

- Photo Avatar API: обучение аватара по фото
- Digital Twin: из фото или короткого видео
- API: $99/мес (100 кредитов), $330/мес (660 кредитов)
- Avatar IV: 6 кредитов/мин (~$3-6/мин)
- 120+ языков, отличный lip-sync
- Лидер рынка AI-видео аватаров

### D-ID (d-id.com) - ДОСТУПНЫЙ ВХОД

- Talking head из одной фотографии
- API: от $18/мес (32 мин streaming)
- Стоимость: ~$0.56-1.13/мин
- 120+ языков, streaming-режим
- Качество ниже HeyGen, но значительно дешевле

### Kling AI (klingai.com) - ВИДЕО С ОБУЧЕНИЕМ

- Custom Models: обучение на 10-30 видео
- Kling 3.0: нативное 4K, 15 сек
- API: $0.07-0.14/сек, через fal.ai $0.09/сек
- Бесплатно: 66 кредитов/день
- Минимальный API-пакет: $4200

### Kohya_ss - ЛУЧШИЙ OPEN SOURCE

- GUI для обучения LoRA (Flux, SDXL, SD 1.5)
- Бесплатно, нужно GPU (от 4GB VRAM)
- На RunPod: ~$1-2 за обучение
- Результат не уступает облачным сервисам при правильных настройках
- Крутая кривая обучения

### SimpleTuner - ПРОДВИНУТЫЙ OPEN SOURCE

- LoRA + full fine-tuning, multi-GPU
- Доминирует в Flux 2 LoRA training
- Бесплатно, нужно 16GB+ VRAM
- pip install simpletuner

## РЕКОМЕНДАЦИИ ПО СЦЕНАРИЯМ

### Фото через API (основной сценарий)

| Приоритет | Сервис | Почему |
|-----------|--------|--------|
| 1 | **Replicate** | Лучший баланс цена/качество/документация. $1.85 обучение, $0.003-0.03 генерация |
| 2 | **fal.ai** | Быстрее inference, Portrait Trainer для лиц. $2.40 обучение |
| 3 | **Astria.ai** | Специализация на лицах, FaceID. Самое дешёвое обучение $1.50 |

### Видео с лицом через API

| Приоритет | Сервис | Почему |
|-----------|--------|--------|
| 1 | **HeyGen** | Лидер, Photo Avatar API, лучший lip-sync. $0.50-0.99/мин |
| 2 | **D-ID** | Доступный вход $18/мес, быстрый результат из фото |
| 3 | **Kling AI** | Custom Models, 4K, через fal.ai $0.09/сек |

### Self-hosted (максимальный контроль)

**Kohya_ss** или **SimpleTuner** + **RunPod** ($0.44/час RTX 4090)
Обучение: ~$2-5 на RunPod. Inference: ComfyUI бесплатно.

### Комбинированный подход (рекомендация)

**Replicate** (фото fine-tuning) + **HeyGen** (видео) + **InsightFace** (face swap на готовые изображения)

## КЛЮЧЕВЫЕ ВЫВОДЫ

1. **Flux - доминирующая технология 2026.** Все топ-сервисы используют Flux от Black Forest Labs.
2. **Обучение стоит $1.50-8.** За 2 минуты. 5-20 фотографий. Порог входа минимальный.
3. **Replicate и fal.ai - лидеры API.** Лучшая документация, экосистема, комьюнити.
4. **HeyGen - лидер видео.** Единственный сервис с полноценным Photo Avatar API.
5. **Open source зрелый.** Kohya_ss + SimpleTuner дают коммерческое качество при наличии GPU.
6. **Российские сервисы отстают** в fine-tuning. Kandinsky 5.0 (MIT) - лучший для self-hosted.
