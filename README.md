# ИНФРАСТРУКТУРА IT

Единая карта всей IT инфраструктуры, инструментов, сервисов и их взаимосвязей.

## НАЗНАЧЕНИЕ

Этот проект содержит:
- **Инвентарь инфраструктуры** - серверы, сети, VPN, DNS
- **Документацию инструментов** - настройки, конфигурации, интеграции
- **Карту взаимосвязей** - как компоненты связаны между собой
- **MCP серверы** - документация всех Model Context Protocol серверов
- **Рекомендации** - как эффективно использовать и развивать инфраструктуру

## ОСНОВНЫЕ ФАЙЛЫ

### `ВСЯ МОЯ ИНФРАСТРУКТУРА (для LLM)__20260201200225.md`

Основной файл с полным инвентарем инфраструктуры:
- Инфраструктура (серверы, сети, VPN, DNS)
- Программы и приложения (Desktop, Mobile, Browser-based)
- Сервисы и SaaS
- AI и автоматизации
- Knowledge-система
- Интеграции и связи

### `docs/tools/`

Детальная документация каждого инструмента:
- **cursor__20260201200225.md** - Cursor IDE (MCP серверы, расширения)
- **warp__20260201200225.md** - Warp терминал (MCP серверы, интеграции)
- **claude-desktop__20260201200225.md** - Claude Desktop (MCP серверы)
- **n8n__20260201200225.md** - n8n (workflows, credentials, интеграции)
- **paddleocr__20260201200225.md** - PaddleOCR (Docker контейнер для OCR)
- **tesseract-ocr__20260201200225.md** - Tesseract OCR (локальный OCR инструмент)

### `docs/mcp-servers__20260201200225.md`

Документация всех MCP (Model Context Protocol) серверов:
- Cursor (10 серверов)
- Warp (5 серверов)
- Claude Desktop (3 сервера)
- Сравнение и статистика

## ПРОЦЕСС ОБНОВЛЕНИЯ

### ДОБАВЛЕНИЕ НОВОГО ИНСТРУМЕНТА

1. Собрать информацию (скриншоты, настройки)
2. Создать документацию в `docs/tools/[название].md`
3. Обновить основной файл `ВСЯ МОЯ ИНФРАСТРУКТУРА (для LLM)__20260201200225.md`
4. Обновить связи в `docs/mcp-servers__20260201200225.md` (если есть MCP серверы)

### ОБНОВЛЕНИЕ СУЩЕСТВУЮЩЕГО ИНСТРУМЕНТА

1. Обновить соответствующий файл в `docs/tools/`
2. Обновить основной файл при необходимости
3. Зафиксировать изменения

## СВЯЗИ С ДРУГИМИ ПРОЕКТАМИ
